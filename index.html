Formatting Rules:
1. Multiple Answers: Separate with a comma (e.g., 0.52,2.62 or pi/6,5pi/6).
2. Exact Values: Use pi (e.g., 3pi/2).
3. Exponents: Use ^2 (e.g., sin^2x).
4. Rounding: For decimal questions, round to exactly 2 decimal places.
